## Előfeltételek
- JDK 21 és Maven telepítés
- Környezeti változók: JAVA_HOME, Path-ra felvenni:  %JAVA_HOME%\bin, maven /bin

## Projekt build
```
mvn install
```

## Projekt futtatás
(a java-client-sample mappában állva)
```
java -jar .\m2m-client\target\m2m-client-1.0.0.jar
```

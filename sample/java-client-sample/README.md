# m2m-java-client-sample projekt

## Fejlesztői leírás
Tutorial: [Tech Tips: Regisztráció aktiválás és token kezelés](https://github.com/nav-gov-hu/M2M/wiki)

## Előfeltételek
- JDK és Maven telepítés
- Környezeti változók: JAVA_HOME, Path-ra felvenni:  %JAVA_HOME%\bin, maven /bin

## Projekt build
```
mvn install
```

## Projekt futtatás
```
java -jar .\target\m2m-client-1.0.0.jar
```
